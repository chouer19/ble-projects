# 关节约定（J1 / J2 / J3）

蜘蛛原型六足：每腿 3 个舵机，行程均为 **0°～180°**。  
连杆：`coxa`（基节）→ `femur`（大腿）→ `tibia`（小腿）→ `tip`（足尖，向下弯曲抓地）。

机体坐标系：原点 = 几何中心，**+X 前、+Y 左、+Z 上**（右手系）。

---

## home = 开机静止姿态

上电后进入的 **合理静止状态**；同时取几何简单的一组角，便于用 `L_c` / `L_f` / `L_t` 做正运动学。

| 关节 | home | 几何 |
|------|------|------|
| J1 | **90°** | 基节沿原点→J1 径向向外 |
| J2 | **180°** | 大腿沿 +Z 竖直向上 |
| J3 | **125°** | 足尖–J3 连线大致与大腿垂直（六腿可分腿标定） |

此时：

- `J2_i = J1_i + L_c · û_i`（`û_i` 为径向单位向量，z = 0）
- `J3_i = J2_i + (0, 0, L_f)`
- 足尖由 J3=125° 与小腿弯折确定（弯折参数后期补）

---

## J1

| 项 | 约定 |
|----|------|
| 作用 | 基节绕竖轴横摆 |
| home | **90°** |
| 角度方向 | 增大 = 绕机体 **+Z** 右手正向（+X → +Y，逆时针） |

---

## J2

| 项 | 约定 |
|----|------|
| 作用 | 大腿俯仰 |
| home | **180°** |
| 角度方向 | 增大 = 从 **腹下 → 头顶**（0° 偏腹下，180° 头顶） |

J1、J2 的机体坐标 **z 均取 0**。

---

## J3

| 项 | 约定 |
|----|------|
| 作用 | 小腿相对大腿转动（足尖固连在小腿上，带向下弯折） |
| **0°** | 小腿抬到最高 |
| **home** | **125°**（tip–J3 ⊥ 大腿，标称） |
| **180°** | 小腿与大腿相触（折叠极限） |
| 角度方向 | 增大 = 从「抬最高」→「贴大腿折叠」 |

### 安装

1. **相触 = 180°**，**抬最高 = 0°**；六腿同一约定。
2. 硬接触前可留软间隙，软件上限略小于 180°。
3. 各腿 home 若偏离 125°，写入分腿标定表覆盖标称值。

## 连杆长度（摘要）

| 符号 | 段 | 长度 |
|------|-----|------|
| `L_c` | J1 → J2（胯/基节） | 0.04 m |
| `L_f` | J2 → J3（大腿） | 0.085 m |
| `L_t` | J3 → tip（小腿） | 0.165 m |
