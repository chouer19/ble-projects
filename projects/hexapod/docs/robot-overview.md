# Hexapod 六足机器人

蜘蛛原型、左右对称的六足步行机构：6 条腿，每腿 3 个关节，共 18 个自由度；全关节采用 Tower Pro MG996R 舵机。单腿串联结构为 **基节（胯）→ 大腿 → 小腿 → 弯折足尖**，无独立踝关节，足尖固连于小腿末端。

---

## 外观与足号

俯视时，六条腿绕机身几何中心分布。**+X 为前方，+Y 为左侧**。

```text
                    +X（前）
                      ↑
           ① 左前          ⑥ 右前
              ╲          ╱
               ╲   机身  ╱
        ② 左中 ●———●———● ⑤ 右中
               ╱         ╲
              ╱           ╲
           ③ 左后          ④ 右后

        ← +Y（左）              −Y（右）→
```

足号：从 **+X 轴** 起，沿 **+Y** 方向逆时针编号 **1～6**。

| 足号 | 位置 |
|------|------|
| 1 | 左前 |
| 2 | 左中（+Y 轴上） |
| 3 | 左后 |
| 4 | 右后 |
| 5 | 右中（−Y 轴上） |
| 6 | 右前 |

对称关系：关于 X 轴，1↔6、2↔5、3↔4；关于 Y 轴，1↔3、6↔4，2 与 5 在 Y 轴上。

## 单腿结构与关节

```text
        机身
          │
    J1 ●──┴── 基节（长 0.040 m，绕 Z 横摆）
          │
    J2 ●────── 大腿（长 0.085 m，俯仰）
          │
    J3 ●────── 小腿（长 0.165 m，膝，相对大腿转动）
          │
          ╰── 足尖（向下弯折，便于抓地）
```

| 关节 | 结构段 | 转轴 | 指令行程 |
|------|--------|------|----------|
| J1 | 基节 / 胯 | 绕机体 +Z | 0°～180° |
| J2 | 大腿 | 与 J1 轴垂直 | 0°～180° |
| J3 | 膝 | 与 J2 轴平行 | 0°～180° |

### 角度正方向

- **J1**：角度增大 = 绕 +Z 逆时针（从 +X 转向 +Y）。
- **J2**：角度增大 = 从腹下转向头顶；0° 偏腹下，180° 为竖直向上。
- **J3**：角度增大 = 从小腿抬最高转向贴大腿折叠；0° 为抬最高，180° 为与大腿相触。

### Home 静止姿态（六腿相同）

| 关节 | Home 角 | 几何含义 |
|------|---------|----------|
| J1 | 90° | 基节沿「原点 → 该腿 J1」径向向外 |
| J2 | 180° | 大腿沿 +Z 竖直向上 |
| J3 | 125° | 足尖与 J3 的连线大致与大腿垂直 |

### J3 安装标定角

| 角度 | 含义 |
|------|------|
| 0° | 小腿抬至最高 |
| 125° | 足尖–J3 连线大致 ⊥ 大腿（标称；各腿可略有差异） |
| 180° | 小腿与大腿相触（折叠极限） |

## 运动学结构

- **J1**：在水平面内改变该腿的基节朝向，从而改变整条腿的工作平面方位。
- **J2、J3**：在上述工作平面内做俯仰与折膝，使足端产生位移；足端轨迹为平面内曲线，在地面上的投影分量取决于当前 J1、J2、J3 角。

各腿 J1 安装位置与方位角不同，因此同一机体运动方向下，六条腿所需的 J1 角并不相同。
